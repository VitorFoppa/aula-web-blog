# 🌐 Meu Blog Pessoal 

Um projeto de portfólio e blog pessoal construído com **HTML**, **CSS** e **JavaScript** puro (Vanilla). Este projeto serve como um espaço para compartilhar meus interesses, hobbies e jornada na tecnologia. Desenvolvido com auxilio do Gemini.Pro.

A grande sacada deste projeto é o seu **sistema de temas duplo**. Ele se apresenta inicialmente com um design "Clean/Pastel", mas esconde um protocolo confidencial no rodapé... 🤫

---

## ✨ Funcionalidades

* **Design Responsivo:** O layout se adapta bem a diferentes tamanhos de tela.
* **Modo Claro (Pastel):** Uma interface limpa, minimalista e aconchegante com tons pastéis, cantos arredondados e sombras suaves.
* **Modo Secreto (Cyberpunk Override):** Um "Easter Egg" ativado por um botão discreto no rodapé. Ele transforma completamente a página com:
    * Animação de transição com efeito *Glitch* (falha de sistema).
    * Paleta de cores Neon (Magenta, Ciano e Verde Hacker).
    * Tipografia de terminal (`Courier New`).
    * Efeito visual de *Scanlines* (linhas de monitor antigo) na tela.
    * Animação de *Jitter* (tremor/oscilação) ao passar o mouse sobre os elementos (Hover).

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização avançada, variáveis, Flexbox e animações complexas (`@keyframes`).
* **JavaScript (ES6):** Manipulação do DOM e controle de classes (`classList`) com `setTimeout` para sincronização da animação de glitch com a troca de tema.

---

## 🚀 Como Executar o Projeto

Como é um projeto *Front-end* estático, você não precisa de nenhuma ferramenta de build ou servidor complexo para rodá-lo localmente.

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/VitorFoppa/aula-web-blog.git](https://github.com/VitorFoppa/aula-web-blog.git)
    ```
2.  Navegue até a pasta do projeto.
3.  Abra o arquivo `index.html` em qualquer navegador da sua preferência.

---

## 📂 Estrutura de Arquivos

* `index.html`: Estrutura principal da página.
* `style.css`: Todo o estilo, incluindo o CSS do modo claro, do modo cyberpunk e das animações.
* `script.js`: A lógica responsável por acionar o modo secreto e controlar o tempo da animação.
* `imagem.jpeg`: Foto de perfil utilizada no cabeçalho.
